# Fonts

## Fonts used in Free League's publications:
### Aniron
User for greater headings.

You can find this free for personal use font on [Pete Klassen's site](https://www.thehutt.de/tolkien/fonts.html)

### New Baskerville
Used for lesser headings and body text.

New Baskerville is one of many revivals of the 1757 Baskerville typeface. A freely licensed substitute is [Baskervville](https://fonts.google.com/specimen/Baskervville)

### Niveau Grotesk
Used for sidebars and tables. You can find this free for personal use font by Hannes von Dohren on [Dafont Free](https://www.dafontfree.io/niveau-grotesk-font/), though it is possible Free League used more typeface features than are available in this free version.

## Kereru
This is the calligraphic font, designed by Daniel Reeve, used for Gandalf's letter at the start of the rulebook. Notably, Reeve did calligraphy for the Peter Jackson films, Rings of Power, the Decipher roleplaying game, and other merchandise. The Italic of this typeface corresponds to what he believes Frodo would write like. Find more information on [Daniel Reeve's website](https://www.danielreeve.co.nz/Fonts/).

### Albertus
Used for the logo. A version of this typeface is included as Flare Serif 821 BT.

### Initials
The books uses initials designed by Niklas Brandt, and they are used as pictures. Included in this repository is a newly created typeface called **InitialRing**, based on these initials. The source file for this file, created with Fontforge, is included in the sources folder.
